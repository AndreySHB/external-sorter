Программа для внешней сортировки строк в больших текстовых файлах.
На первом этапе файл дробиться на несколько файлов размера не больше заданного, содержимое которых отсортировано в памяти.
На втором этапе происходит сортировка через промежуточные файлы методом попарного слияния.

Сортировка запускается через метод main класса Main.

Первый аргумент программы - путь к каталогу, где лежит файл для сортировки.

Второй - имя файла для сортировки.

Третий - имя отсортированного файла, который будет создан.

Четвертый - максимальное число строк, которое может быть помещено в оперативную память для сортировки

Если файла не существует, то он будет сгенерирован со следующими параметрами:

Общее количество строк - пятый аргумент

Максимальный размер строки в битах - шестой аргумент

Время выполнения сортировки при некоторых параметрах (i5 6600K, HDD 7200 prm, максимальная длина строки 100 бит):

1) NUM_STRINGS = 1_000_000, BATCH_SIZE = NUM_STRINGS/10: 23s (размер файла ~ 20 мб)
2) NUM_STRINGS = 10_000_000, BATCH_SIZE = NUM_STRINGS/10: 248s (размер файла ~ 200 мб)
3) NUM_STRINGS = 100_000_000, BATCH_SIZE = NUM_STRINGS/10: 2489s (размер файла ~ 2000 мб)

(Если BATCH_SIZE = CONST, то время сортировки возрастало бы как nlog(n), из-за увеличения высоты дерева)
